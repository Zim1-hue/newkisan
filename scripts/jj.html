\chapter{Abstract} \onehalfspacing This project presents the development of a software-based agricultural decision support system called Indian Farmer Assistant, designed to improve crop selection and farm management using data-driven techniques. The system utilizes a Random Forest Machine Learning model trained on an agricultural dataset to predict the most suitable crops based on environmental and soil parameters such as temperature, rainfall, humidity, and nutrient values (N, P, K). In addition to crop prediction, the system estimates expected crop yield (kg/hectare) and recommends alternative crops with simple reasoning. It also includes an image-based plant disease detection module to identify crop diseases and suggest basic treatments. To enhance usability, the system provides farmer-friendly reports summarizing predictions, recommendations, and advisory insights. The application integrates automatic weather data to improve prediction accuracy and reduce manual input effort. It also offers basic farming guidance such as irrigation and nutrient suggestions using rule-based logic. Furthermore, the system supports multiple languages, including English, Hindi, and Bengali, ensuring accessibility for a wider range of users. Overall, the proposed system combines Machine Learning with a simple and user-friendly interface to assist farmers in making better agricultural decisions, improving productivity, and reducing risks associated with climate variability.
